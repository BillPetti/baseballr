# **(legacy) Scrape Batter Performance Data Over a Custom Time Frame**

**(legacy) Scrape Batter Performance Data Over a Custom Time Frame**

## Usage

``` r
daily_batter_bref(t1, t2)
```

## Arguments

- t1:

  First date data should be scraped from. Should take the form
  "YEAR-MONTH-DAY"

- t2:

  Last date data should be scraped from. Should take the form
  "YEAR-MONTH-DAY"

## Value

Returns a tibble of batter performance
