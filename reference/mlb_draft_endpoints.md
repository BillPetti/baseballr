# **MLB Draft Endpoint Overview**

- [`mlb_draft()`](https://billpetti.github.io/baseballr/reference/mlb_draft.md):
  Retrieve draft pick information by year.

- [`mlb_draft_latest()`](https://billpetti.github.io/baseballr/reference/mlb_draft_latest.md):
  Retrieve latest draft information by year.

- [`mlb_draft_prospects()`](https://billpetti.github.io/baseballr/reference/mlb_draft_prospects.md):
  Retrieve draft prospect information by year.

## Details

### **MLB Draft**

These functions retrieve MLB Draft pick, latest pick, and prospect
information from the MLB Stats API.
