# **(legacy) Download a data frame of supplemental data about MLB games since 2008.**

**(legacy) Download a data frame of supplemental data about MLB games
since 2008.**

## Usage

``` r
get_game_info_sup_petti()
```

## Value

Function returns a tibble with various columns, including:

- game_pk

- game_date

- venue id

- attendance

- game temperature

- wind speed

- direction

- start time

- end time
