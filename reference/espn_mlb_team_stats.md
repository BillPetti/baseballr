# **Get ESPN MLB team stats data**

**Get ESPN MLB team stats data**

## Usage

``` r
espn_mlb_team_stats(team_id, year, season_type = "regular", total = FALSE)
```

## Arguments

- team_id:

  Team ID

- year:

  Year

- season_type:

  (character, default: regular): Season type - regular or postseason

- total:

  (boolean, default: FALSE): Totals
