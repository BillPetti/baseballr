# **Most Recent College Baseball Season**

NCAA college baseball seasons run roughly February through June (the
College World Series finishes in late June) and are labelled by the year
in which they end. This returns the current season's year from February
onward, and the prior year in January (before the season opens).

## Usage

``` r
most_recent_college_baseball_season()
```

## Value

An integer indicating the year of the most recent NCAA college baseball
season.

## Examples

``` r
most_recent_college_baseball_season()
#> [1] 2026
```
