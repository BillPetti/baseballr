# **MLB Stats Endpoint Overview**

- [`mlb_stats()`](https://billpetti.github.io/baseballr/reference/mlb_stats.md):
  MLB Stats.

- [`mlb_stats_leaders()`](https://billpetti.github.io/baseballr/reference/mlb_stats_leaders.md):
  MLB Stats Leaders.

- `mlb_stats_streaks()`: MLB Stats Streaks.

- [`mlb_stat_groups()`](https://billpetti.github.io/baseballr/reference/mlb_stat_groups.md):
  MLB Stat Groups.

- [`mlb_stat_types()`](https://billpetti.github.io/baseballr/reference/mlb_stat_types.md):
  MLB Stat Types.

- [`mlb_baseball_stats()`](https://billpetti.github.io/baseballr/reference/mlb_baseball_stats.md):
  MLB Baseball Stats.

## Details

### **MLB Stats**

These functions retrieve MLB statistics, leaders, streaks, stat groups,
stat types, and baseball stat metadata from the MLB Stats API.
