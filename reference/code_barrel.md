# **Helper for determining whether a batted ball is a "barrel"**

This function allows you to code a batted ball as a barrel as defined by
the Statcast research team using data as provided by
baseballsavant.mlb.com.

## Usage

``` r
code_barrel(df)
```

## Arguments

- df:

  A dataframe generated by baseballsavant.mlb.com that must include the
  following variables: `launch_angle` and `launch_speed`.

## Value

Returns a tibble with the additional column, `barrel`.
