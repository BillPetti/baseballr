# **MLB Schedule Endpoint Overview**

- [`mlb_schedule()`](https://billpetti.github.io/baseballr/reference/mlb_schedule.md):
  Find game_pk values for professional baseball games (major and minor
  leagues).

- [`mlb_schedule_postseason()`](https://billpetti.github.io/baseballr/reference/mlb_schedule_postseason.md):
  Find game_pk values for professional baseball postseason games (major
  and minor leagues).

- [`mlb_schedule_postseason_series()`](https://billpetti.github.io/baseballr/reference/mlb_schedule_postseason_series.md):
  Find game_pk values for professional baseball postseason series games
  (major and minor leagues).

- [`mlb_schedule_games_tied()`](https://billpetti.github.io/baseballr/reference/mlb_schedule_games_tied.md):
  Find game_pk values for professional baseball games (major and minor
  leagues) that are tied.

- [`mlb_schedule_event_types()`](https://billpetti.github.io/baseballr/reference/mlb_schedule_event_types.md):
  MLB Schedule Event Types.

## Details

### **MLB Schedule**

These functions retrieve MLB schedule, postseason, tied-game, and
event-type information from the MLB Stats API.
