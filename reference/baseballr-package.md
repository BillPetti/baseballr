# baseballr: Acquiring and Analyzing Baseball Data

Provides numerous utilities for acquiring and analyzing baseball data
from online sources such as 'Baseball Reference'
<https://www.baseball-reference.com/>, 'FanGraphs'
<https://www.fangraphs.com/>, and the 'MLB Stats' API
<https://www.mlb.com/>.
